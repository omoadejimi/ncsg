## Principles

### 1. Source text must be plain first

Idioms, jokes and corporate English fail in translation. Fix the English with [plain language](/docs/foundations/plain-language) before translating. This reduces translation cost and error.

### 2. Use professional speakers, review with users

Ensure you test translations with actual speakers before you publish.


### 3. Pidgin is a full language

Nigerian Pidgin reaches across regions and literacy levels. Treat it with the same rigour as other Nigerian languages.


### 4. Plan for expansion

Design buttons and layouts for +35% text length, and never truncate a warning to fit.
