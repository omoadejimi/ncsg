## Default: sentence case

Use sentence case for headings, buttons, labels, menu items and notifications. Capitalise only the first word and proper nouns.

<DoDont>
<Do>
Send money to a new recipient
</Do>
<Dont>
Send Money To A New Recipient
</Dont>
</DoDont>

Sentence case is easier to read, easier to translate, and removes daily debates about which words "deserve" capitals.

## Proper nouns and product terms

- Capitalise real names: **Lagos**, **First Bank**, **NIBSS**, **Ramadan**.
- Capitalise official scheme names: **Bank Verification Number (BVN)** on first use, **BVN** after.
- Don't capitalise generic features to make them sound important: it's "your wallet", not "your Wallet".

## Never use ALL CAPS

Not for emphasis, not for headings, not for legal text. It reads as shouting, slows reading, and can be mishandled by screen readers. If something must stand out, use weight or layout.
